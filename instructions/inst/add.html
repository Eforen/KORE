1.2.1. ADD (Add Registers)¶

Add one register (rs1) to another (rs2) and store the result it in a 3rd destination register (rd)

Note: Arithmetic overflow is ignored.

Formula \(x[rd] \leftarrow x[r_1] + x[r_2]\)

ADD rs, r1, r2
